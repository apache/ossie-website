## Working Groups

The OSI initiative is organized into focused working groups, each tackling a specific area of the specification:

| Working Group | Focus Area |
|---|---|
| Advanced Metrics & Expression Language | Defining complex metric calculations and a portable expression syntax |
| Composability | Enabling semantic models to reference and extend one another |
| Catalog Integration | Bridging OSI models with data catalog and governance platforms |
| Ontology Representation | Mapping OSI concepts to formal ontology standards |
| Model Converters & Developer Tools | Building tooling for importing, exporting, and validating OSI models |

## Code of Conduct

All participants in the OSI community are expected to follow our code of conduct. We are committed to providing a welcoming, inclusive, and harassment-free experience for everyone.
